HOW DO I CONTRIBUTE?

Step 1

Go to the github organization page and select a repository/project that you would like to contribute

https://github.com/Mozilla-Campus-Club-IIT

Step 2

Take a look at the issues

Step 3

Fork the project to you account by clicking the 'Fork' button

Step 4

Clone the project to your local storage by copying the repo url and running the following command in the terminal



Commad : git clone url-of-repo.git

Step 5

Open project in your favourite IDE/Text editor and make the changes

Step 6

Add and commit the changes with the following commands

Add Commad : git add . | Commit Commad : git commit -m "your commit message comes here"

Step 7

Push your local repo chamges to your remote repo using the follwing cammand

Commad : git push

Step 8

Finally , make a pull request to send your changes to the main organization repo

Thats it :)

Now you know how to contribute to an organization in Github .
